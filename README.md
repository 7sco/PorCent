<h3 align="center">PorCent</h3>

Percentage calculator made with Jetpack Compose

## Feature
- Material You design
- Adaptive Colors
- Adaptive Icon
- <a href="https://developer.android.com/jetpack/compose/glance">Jetpack Glance Widget</a>
